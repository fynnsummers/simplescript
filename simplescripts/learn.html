Erklärungen der Befehle

  • set [Variable] [Wert]: Setzt eine Variable auf einen bestimmten Wert.
  • add [Variable] [Wert]: Addiert einen Wert zu einer bestehenden Variable.
  • subtract [Variable] [Wert]: Subtrahiert einen Wert von einer bestehenden Variable.
  • multiply [Variable] [Wert]: Multipliziert den Wert einer bestehenden Variable mit einem angegebenen Wert.
  • divide [Variable] [Wert]: Dividiert den Wert einer bestehenden Variable durch einen angegebenen Wert.
  • mod [Variable] [Wert]: Setzt die Variable auf den Restwert der Division durch den angegebenen Wert.
  • power [Variable] [Wert]: Setzt die Variable auf die Potenz der angegebenen Zahl.
  • random [Variable] [min] [max]: Setzt die Variable auf eine zufällige Zahl zwischen min und max.
  • reset [Variable]: Setzt die angegebene Variable auf 0.
  • print [Text oder Variable]: Gibt den Text oder den Wert einer Variable aus. Kann auch mehrere Variablen im Format print a ist a und b ausgeben.
  • wait [Sekunden]: Wartet für die angegebene Anzahl an Sekunden, bevor der nächste Befehl ausgeführt wird.
  • increase_every [Sekunden] [Wert] [Variable]: Erhöht eine Variable alle angegebenen Sekunden um den bestimmten Wert.
  • repeat [Anzahl]: Wiederholt den folgenden Befehl eine bestimmte Anzahl von Malen.
  • if [Variable] > [Wert] then: Führt eine Aktion aus, wenn die Bedingung wahr ist.
  • stop: Beendet die Ausführung des Skripts.

Beispiel-Skripte

Beispiel 1: Bedingte Ausgabe

    set zahl 10
    if zahl > 5 then
        print "Die Zahl ist größer als 5"
    stop

Ausgabe:

    Die Zahl ist größer als 5
    Ende

Beispiel 2: Modulo und Potenz

    set num 10
    mod num 3
    print num
    power num 2
    print num
    stop

Ausgabe:

    1
    1
    Ende

Beispiel 3: Addition und Warten

    set a 5
    add a 3
    print a ist a
    wait 2
    print Warte 2 Sekunden.
    stop

Ausgabe:

    a ist 8
    Warte 2 Sekunden.
    Ende

Beispiel 4: Zufallszahl und Reset

    random d 1 100
    print d ist d
    reset d
    print d ist jetzt d
    stop

Ausgabe:

    d ist [Zufallszahl]
    d ist jetzt 0
    Ende

Beispiel 5: Addition von Variablen

    set a 10
    set b 20
     a + b = c
    print 10 Plus 20 ist 'c'
    stop

Ausgabe:

    10 Plus 20 ist 30
    Ende
